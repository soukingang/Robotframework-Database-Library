Robotframework-Database-Library by franz-see

Database Library contains utilities meant for Robot Framework's usage. This can allow you to query your database after an action has been made to verify the results. This is compatible* with any Database API Specification 2.0 module.

Requirements

  • Python / Jython / IronPython
  • Robotframework
  • A Database API Specification 2.0 Python Module that you will use to connect to the database.

Install

Using easy_install

easy_install robotframework-databaselibrary

Using pip

pip install robotframework-databaselibrary

From Source

Download source from https://github.com/franz-see/Robotframework-Database-Library/archives/0.2. Extract the tarball or the zip file then enter the extracted directory. Then run 
python setup.py install
.
